## 🚀 Subir a GitHub Pages (GRATIS)

1. Crea una cuenta en [github.com](https://github.com) si no tienes
2. Crea un nuevo repositorio llamado `ferroprime` (público)
3. Sube todos los archivos (arrastra y suelta en GitHub)
4. Ve a **Settings → Pages**
5. En "Source" selecciona `main` branch y carpeta `/root`
6. Haz clic en **Save**
7. Tu sitio estará en: `https://TU_USUARIO.github.io/ferroprime`

## 🌐 Asignar tu dominio propio

Una vez que tengas GitHub Pages activo:
1. En **Settings → Pages → Custom domain**, escribe tu dominio
2. En tu proveedor de dominio, crea un registro DNS:
   - Tipo: `CNAME`
   - Nombre: `www`
   - Valor: `TU_USUARIO.github.io`
3. ¡Listo! En 24-48h tu dominio apuntará al sitio.
